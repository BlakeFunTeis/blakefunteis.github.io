ES6 新特性

ES6的目標是使JavaScript可以開發更複雜的大型應用，並成為企業級開發語言。

ES6中新增了let和const這兩種用來宣告的關鍵字，我們來試試看他們與我們以前常用的var有什麼不同。

let

let跟var的差別是var在宣告時會把自己提升到程式頂端。

這樣用var宣告，他不會報錯，但是如果換成let就會報錯。

const

const比較特殊的是如果是「值」的宣告，那麼他在後面就不能再改變，但是如果是陣列或物件，那麼陣列和物件內的值還是可以做改變，這是因為const的宣告常數是指向該物件或陣列的記憶體位置，只要記憶體位置不變，那麼就不會錯誤。

展開運算子
